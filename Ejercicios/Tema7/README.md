<h2>Ejercicio 7.1</h2>

Escribir un programa que genere un camino aleatorio en dos dimensiones, en el que los pasos sean de longitud unidad, con el ángulo aleatorio. Calcular el valor promedio de r2 para 500 caminos de 100 pasos. Obtener el valor de D mediante un ajuste por mínimos cuadrados.

Modificar el programa anterior para permitir pasos de longitud variable entre 0.5 y 2 unidades. Comprobar el valor de D en este caso.

Incluir junto con el código correspondiente un documento pdf con los gráficos del promedio de r2 frente al número de pasos y los valores de D obtenidos para cada caso.
